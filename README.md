# 微信小程序：百味博客音乐播放器

该小程序为针对light-music提供的源码进行优化改进
* 修复播放时间会出现毫秒显示问题
* 修复分享音乐播放问题
* 增加自动连播功能
* 优化界面列表导航位置到左边，并增加列表返回按钮（原右边和微信小程序自带的按钮重叠）
* 使用全屏显示播放器
* 修改播放器风格
* 增加使用api获取音乐列表

感谢原作者：https://github.com/facejiong/light-music-wechat-mini-programs
